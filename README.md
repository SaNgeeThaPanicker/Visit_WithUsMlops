# Visit With Us — MLOps Pipeline

## Project Overview
An end-to-end MLOps pipeline to predict whether a customer will purchase 
the Wellness Tourism Package.

## Pipeline Steps
1. Data Registration → Hugging Face Dataset Hub
2. Data Preparation → Clean, encode, split, re-upload
3. Model Training → Random Forest + Gradient Boosting with MLflow tracking
4. Model Deployment → Streamlit app on Hugging Face Spaces
